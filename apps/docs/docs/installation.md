```shell
# With pnpm
pnpm add express-cargo reflect-metadata

# Or with npm
npm install express-cargo reflect-metadata

```

## Requirements

- Node.js 20+
- TypeScript enabled


### tsconfig.json
```json
{
    "compilerOptions": {
        "experimentalDecorators": true,
        "emitDecoratorMetadata": true
    }
}
```
